# b-text 文本块
> 用于显示普通文本，可设置文本字体大小和颜色

### 属性列表
> 注意：文字大小是按 UI 视觉稿上的标注来定的，比如 22px 等

 属性 | 说明 | 备注 
--- | --- | ---
 size | 文字大小 | 单位为视觉稿中的 px 像素 
 color | 文字颜色 | 不可填写颜色值，目前可选的颜色有：lightgrey、grey、black、highlight 等

### 子节点
> 纯文本内容

### 示例
```
<b-text size='34' color='black'>手机号码</b-text>
```
