# b-select 选择框
> 用于进行选择某一项

### DEMO
![DEMO](https://ohc0dpsgs.qnssl.com/image/service/serviceBanner.jpg)

### 属性列表
> 注意：

 属性 | 说明 | 备注 
--- | --- | ---
 size | 文字大小 | 单位为视觉稿中的 px 像素 
 color | 文字颜色 | 不可填写颜色值，目前可选的颜色有：lightgrey、grey、black、highlight 等

### 子节点
> 纯文本内容

### 示例
```
<b-select size=".75" >身份证</b-select>
```
