# a-button 按钮节点
> 声明每个按钮组件中的按钮节点

### DEMO
<div><img alt="DEMO" src="https://ohc0dpsgs.qnssl.com/lego/images/button.jpeg" width="280.859"/></div>

### 属性列表

属性 | 说明 | 可选值 | 备注 
--- | --- | --- | ---
id | 节点标识 | 符合 id 规范的字符串 | 用于在 controller 中绑定 tap 事件


### 子节点
>  文本，即按钮上的文字

### 示例
```
<dock bgcolor='white'>
    <a-button id='cancel'>取消</a-button>
    <a-button id='ok'>确定</a-button>
</dock>
```
